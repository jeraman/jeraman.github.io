Steal & Repeat Machine

27 Apr, 2017
Code Video

The Steal & Repeat Machine is a live looping tool based on direct manipulation of sound.

The Steal & Repeat Machine consists of an iPad physically extended via a plexiglass case equipped with an external sound card and two knob controls for phasing and feedback.

The iPad runs a software interface implemented in C++/Openframeworks, presented below:

  • Circuits;

  • Powering;
